# 어문규범 점검 (맞춤법·띄어쓰기·문장부호)

> 근거: 국립국어원 공공 어문규범(「한글 맞춤법」, 「표준어 규정」, 「문장 부호」 규정 — 공공저작물). 아래 예문은 모두 창작된 가상 예시(가상의 원고·피고·계약·증거)로, 특정 사건이나 인물과 무관하다.

법률문서 초안을 다듬을 때 자주 틀리는 어문규범을 빠르게 점검하기 위한 체크리스트다. 각 항목은 **규칙 1~2문장 + ✗→✓ 오리지널 예문**으로 구성된다.

---

## A. 띄어쓰기

### A-1. 의존명사는 띄어 쓰고, 같은 형태의 조사·어미는 붙여 쓴다

의존명사(것·수·바·데·지·시·때·만큼·뿐·대로·터·줄 등)는 앞말과 띄어 쓴다. 다만 같은 글자가 조사나 어미로 쓰일 때는 앞말에 붙인다. 의미로 구분한다.

| | ✗ | ✓ |
|---|---|---|
| 것(의존명사) | 피고가 변제할것을 약정하였다 | 피고가 변제할 것을 약정하였다 |
| 수(의존명사) | 이행을 강제할수 없다 | 이행을 강제할 수 없다 |
| 시(時, 의존명사) | 판결선고시 / 변제기 도래시 | 판결 선고 시 / 변제기 도래 시 |
| 때(의존명사) | 진료할때마다 | 진료할 때마다 |
| 바(의존명사) | 앞서 살펴본바와 같다 | 앞서 살펴본 바와 같다 |
| -ㄴ바(어미) | 증거를 검토한 바, 위조로 보인다 | 증거를 검토한바, 위조로 보인다 |
| 데(의존명사, 곳·경우) | 손해를 산정하는데 어려움이 있다 | 손해를 산정하는 데 어려움이 있다 |
| -는데(어미) | 계약은 유효한 데, 이행이 없다 | 계약은 유효한데, 이행이 없다 |
| 지(시간 경과) | 계약을 체결한지 6개월이 지났다 | 계약을 체결한 지 6개월이 지났다 |
| -ㄴ지(어미, 막연) | 이행이 가능한 지 의문이다 | 이행이 가능한지 의문이다 |
| 만큼(의존명사) | 청구한만큼 인용되어야 한다 | 청구한 만큼 인용되어야 한다 |
| 만큼(조사) | 원고 만큼 성실한 채무자도 없다 | 원고만큼 성실한 채무자도 없다 |
| 뿐(의존명사) | 통보하였을뿐 이행은 없었다 | 통보하였을 뿐 이행은 없었다 |
| 뿐(조사) | 증거는 차용증 뿐이다 | 증거는 차용증뿐이다 |
| 대로(의존명사) | 약정한대로 지급하라 | 약정한 대로 지급하라 |
| 대로(조사) | 계약서 대로 이행하라 | 계약서대로 이행하라 |

- 시간 경과의 '지'는 띄우고('체결한 지 6개월'), 막연한 의문의 '-ㄴ지/-는지'는 어미이므로 붙인다('가능한지 의문이다').
- '데'가 '곳·경우·일'을 뜻하면 의존명사(띄움), 연결·배경을 나타내면 어미 '-는데'(붙임).

### A-2. 단위명사·숫자, 금액의 한글 표기

수를 나타내는 말 뒤의 단위명사(원·명·건·개월·점 등)는 띄어 쓰는 것이 원칙이다. 다만 순서나 아라비아 숫자와 어울려 쓰일 때는 붙여 쓸 수 있다. 금액(수)을 한글로 적을 때는 '만(萬)' 단위로 띄어 쓴다.

| | ✗ | ✓ |
|---|---|---|
| 단위명사(원칙) | 손해배상금 3000만원 | 손해배상금 3000만 원 |
| 단위명사(명·건) | 증인 3명, 관련 사건 5건 (붙임) | 증인 3 명 → 증인 3명 / 5 건 → 5건 |
| 기간 | 변제기까지 6 개월 | 변제기까지 6개월 |
| 숫자+단위 허용 | — | 제3 항(원칙) / 제3항(허용) |
| 금액 한글표기 | 금일억오천만원 | 금 일억 오천만 원 |
| 금액 한글표기 | 일금삼천이백오십만원정 | 일금 삼천이백오십만 원정 |

- 아라비아 숫자 뒤 단위는 붙여 쓰는 것이 실무 관행으로 허용된다('3명', '5건', '6개월'). 한글 수사 뒤 단위명사는 원칙적으로 띄운다('세 명', '다섯 건').
- 금액은 큰 단위(억·만)에서 띄우되, '원정(整)'은 위조·변조 방지용 관용 표기.

### A-3. '제(第)+숫자'는 붙이고, 차례를 나타내는 말

순서·차례를 나타내는 접두사 '제(第)'는 뒷말에 붙여 쓴다. '제'가 붙은 말과 단위명사는 붙여 쓰는 것이 허용된다.

| | ✗ | ✓ |
|---|---|---|
| 제+숫자 | 제 1심 판결 | 제1심 판결 |
| 제+조항 | 제 12 조 제 3 항 | 제12조 제3항 (또는 제12조제3항) |
| 차례 | 갑 제 3 호증 | 갑 제3호증 |
| 순서 | 제 2 차 변론기일 | 제2차 변론기일 |

- '제1심', '제3자', '제2항'처럼 '제'는 항상 붙인다. 조·항·호 사이는 띄움이 원칙이나 법령 인용에서는 '제12조제3항'처럼 붙여 쓰는 관행도 통용된다(일관성 유지가 핵심).

### A-4. 보조용언 띄어쓰기 — 원칙은 띄움, 경우에 따라 붙임 허용

보조용언은 띄어 쓰는 것이 원칙이나, 본용언에 붙여 쓰는 것도 허용된다(앞말이 합성동사이거나 중간에 조사가 들어가면 띄어 써야 한다).

| | 원칙(✓) | 허용(✓) | 주의(✗) |
|---|---|---|---|
| -아/어 + 보조용언 | 검토해 보았다 | 검토해보았다 | — |
| -아/어 + 두다 | 보관해 두었다 | 보관해두었다 | — |
| -지 + 못하다/아니하다 | 이행하지 못하였다 | 이행하지못하였다(붙임 비권장) | — |
| 합성어 뒤 | — | — | 검토해봐 두었다 |
| 조사 개입 | 검토해는 보았다 | — | 검토해는보았다 |

- 안전하게 가려면 **원칙대로 띄어 쓴다**('밝혀 둔다', '제출해 두었다', '지급하지 아니하였다').
- 다만 띄어쓰기와 별개로, **윤문 단계에서는 '아니하다'를 평이한 준말 '않다'로 바꾼다**('지급하지 아니하였다'→'지급하지 않았다', '~하지 아니한 채'→'~하지 않은 채'). 어문규범상 '아니하다'가 틀린 것은 아니지만, 평이화 대상이다(principles.md §4 어휘 평이화 표 참조).

### A-5. 열거의 '등'(의존명사, 띄움) vs 복수접미사 '들'(붙임)

여러 가지를 열거한 뒤의 '등(等)'은 의존명사이므로 띄어 쓴다. 반면 복수를 나타내는 접미사 '들'은 앞말에 붙여 쓴다.

| | ✗ | ✓ |
|---|---|---|
| 등(의존명사) | 차용증, 입금내역등 증거 | 차용증, 입금내역 등 증거 |
| 들(접미사) | 피고 들이 공모하여 | 피고들이 공모하여 |
| 들(의존명사, 그것들) | 증인 들이 일치하여 진술 | 증인들이 일치하여 진술 |

- '등등', '등지'도 의존명사로 띄운다('서울, 부산 등지에서').

### A-6. 관형사 '이/그/저'는 띄어 쓴다 — '이 사건' 등

지시관형사 '이·그·저'는 뒤의 명사와 띄어 쓴다. 다만 '이때·그때·이번·이놈' 등 한 단어로 굳어진 합성어는 붙인다.

| | ✗ | ✓ |
|---|---|---|
| 이 + 명사 | 이사건 계약은 무효이다 | 이 사건 계약은 무효이다 |
| 그 + 명사 | 그계약의 효력 | 그 계약의 효력 |
| 저 + 명사 | 저증거의 신빙성 | 저 증거의 신빙성 |
| 합성어 | 이 때 비로소 | 이때 비로소 |
| 합성어(한 단어) | 오래 전부터 / 그 동안 / 한 데 묶어 | 오래전부터 / 그동안 / 한데 묶어 |
| 접미사 붙임 | 1주일 가량 / 10여 명 / 한 달 쯤 | 1주일가량 / 10여 명 / 한 달쯤 |

- '가량·쯤·여(餘)'는 수량·정도를 더하는 **접미사**이므로 앞말에 붙인다('2개월가량', '30분쯤'). 단 '여'는 '10여 명'처럼 뒤 단위명사와는 띄운다('10여∨명').

### A-7. 법령명·전문용어 띄어쓰기 — 원칙(띄움) / 허용(붙임)

전문용어는 단어별로 띄어 씀이 원칙이나, 붙여 쓸 수 있다. 법령명은 고유명사에 준하여 띄어 쓰되, 약칭·관용 표기는 통일성을 유지한다.

| | 원칙(✓) | 허용(✓) |
|---|---|---|
| 법령명 | 민법 제750조 | (동일) |
| 전문용어 | 채무 불이행 책임 | 채무불이행책임 |
| 전문용어 | 손해 배상 청구권 | 손해배상청구권 |
| 죄명 | 사기 미수죄 | 사기미수죄 |

- 한 문서 안에서는 **한 가지 방식으로 통일**한다(섞어 쓰지 않는다).

### A-8. 자주 틀리는 '안 되다/안되다', '및', '내/외/상/하'

부정의 '안'(=아니)은 띄어 쓰고, '안되다'(=잘되지 않다, 가엾다)는 한 단어로 붙인다. '및'은 앞뒤를 띄운다. '내·외·상·하'가 명사 뒤에서 의존명사·접사로 쓰일 때의 띄어쓰기에 주의한다.

| | ✗ | ✓ |
|---|---|---|
| 안(부정) | 변제가 안된다 → 안 됨이 명백 | 변제가 안 된다 / 입증이 안 되었다 |
| 안되다(한 단어) | 사정이 안 되어 보인다(가엾다) | 사정이 안되어 보인다 |
| 및 | 원고및피고는 | 원고 및 피고는 |
| 내(의존명사) | 기간내에 이행하라 | 기간 내에 이행하라 |
| 외(의존명사) | 그 외에도 | 그 외에도 (그밖에 → 그 밖에) |
| 상(접사) | 계약 상 의무 | 계약상 의무 |
| 하(접사) | 감독 하에 | 감독하에 |

- '상·하·중·내·외'가 추상적 관계·조건을 뜻하는 접사일 때는 붙인다('법률상', '사실상', '진행 중' vs '회의 중'은 의미에 따라). '관계상', '업무상'은 붙임. '기간 내', '건물 내'처럼 공간·범위의 '내'는 띄움이 일반적.
- '안 된다'(부정)와 '안된다'(가엾다·곤란하다)는 의미로 구별: "그 정도 손해는 안됐다"(가엾다) vs "변제가 안 됐다"(이루어지지 않았다).

## B. 맞춤법·표기

### B-1. 자주 틀리는 표기

의미·문법 기능으로 구별해야 하는 대표적 혼동 사례. 법률문서 톤의 오리지널 예문으로 정리한다.

| 구분 | ✗ | ✓ | 메모 |
|---|---|---|---|
| 되/돼 | 변제기가 도래하면 변재돼야 한다 | 변제기가 도래하면 변제되어야 한다 | '돼'는 '되어'의 준말일 때만 |
| 되/돼 | 청구가 인용되여 | 청구가 인용돼(=되어) | 문장 끝 '~돼'(O), '~되'(X) |
| -로서/-로써 | 대리인으로써 출석하였다 | 대리인으로서 출석하였다 | 자격·지위=로서 |
| -로서/-로써 | 차용증으로서 입증한다 | 차용증으로써 입증한다 | 수단·도구=로써 |
| -든지/-던지 | 이행을 하던지 말던지 | 이행을 하든지 말든지 | 선택=든지 |
| -든지/-던지 | 그때 얼마나 독촉했든지 | 그때 얼마나 독촉했던지 | 과거 회상=던지 |
| 율/률 | 지연손해금 비율은 연 12프로 | 지연손해금 이율은 연 12% | 받침 없거나 ㄴ받침=율, 그 외=률 |
| 율/률 | 출석율, 합의률 | 출석률, 합의율 | 출석(ㄱ받침)→률 / 합의(받침X)→율 |
| 결재/결제 | 대금을 결재하였다 | 대금을 결제하였다 | 돈=결제 / 승인=결재 |
| 갱신/경신 | 계약 기간을 경신하였다 | 계약 기간을 갱신하였다 | 법률관계 연장=갱신 / 기록 깨기=경신 |
| 채/체/째 | 도장을 찍은 체 / 통째로 → 통채로 | 도장을 찍은 채 / 통째로 | 상태유지=채, 척=체, 전부/순서=째 |
| 가르치다/가리키다 | 증거가 피고를 가르킨다 | 증거가 피고를 가리킨다 | 지시=가리키다 |
| 맞히다/맞추다 | 답을 맞추다(정답) | 답을 맞히다(정답) | 정답=맞히다 / 비교·조정=맞추다 |
| 일절/일체 | 책임을 일절 부담한다 | 책임을 일체 부담한다 | 모두=일체 / 전혀(부정)=일절 |
| 부치다/붙이다 | 우편으로 붙이다 | 우편으로 부치다 | 보내다=부치다 / 접착=붙이다 |
| 받치다/받히다 | 차에 받쳐 → | 차에 받혀(부딪힘) | 피동(부딪힘)=받히다 |
| -에/-의 | 원고에 주장 | 원고의 주장 | 소유·관계=의 |
| 웬/왠 | 왠 서류가 첨부 | 웬 서류가 첨부 | '왜인지'=왠지만, 나머지=웬 |
| 빌리다/빌려 | 이 자리를 빌어 | 이 자리를 빌려 | 표준어=빌리다 |
| 며칠 | 몇일 후 | 며칠 후 | '며칠'이 표준 |

### B-2. 한자 병기 관행(괄호)

뜻을 분명히 할 필요가 있는 용어는 한글로 적고 괄호 안에 한자를 병기한다(한글—한자 사이를 띄우지 않는다). 동음이의어·전문용어 구별, 인명·금액 위·변조 방지 목적.

| | ✗ | ✓ |
|---|---|---|
| 병기 형식 | 경개 (更改) 의 의사 | 경개(更改)의 의사 |
| 띄어쓰기 | 변제 ( 辨濟 ) | 변제(辨濟) |
| 조사 처리 | 채무(債務) 을 | 채무(債務)를 ← 한글 '채무'의 받침 기준 |

- 괄호 뒤 조사는 **한글 본말의 받침**을 기준으로 선택한다('채무(債務)를', '경개(更改)는').

## C. 문장부호(국립국어원 「문장 부호」)

### C-1. 가운뎃점·쉼표·붙임표

가운뎃점(·)은 짝을 이루는 어구를 묶거나 같은 계열의 항목을 나열할 때 쓴다. 쉼표(,)는 같은 자격의 어구를 열거하거나 문장 안에서 끊어 읽을 자리에 쓴다. 붙임표(-)는 차례·범위·연결을 나타낸다.

| | ✗ | ✓ | 메모 |
|---|---|---|---|
| 가운뎃점 | 원고, 피고간 합의 | 원고·피고 간 합의 | 짝을 이루는 어구 묶음 |
| 가운뎃점 | 민법, 상법, 형법상 | 민법·상법·형법상 | 같은 계열 나열 |
| 쉼표(열거) | 차용증 입금내역 문자메시지를 | 차용증, 입금내역, 문자메시지를 | 같은 자격 열거 |
| 붙임표(범위) | 2021년~2023년분 | 2021-2023년분 / 2021~2023년 | 물결표(~)도 범위 표시 가능 |

- 접속부사(그러나·따라서·그러므로·또한·다만) 뒤의 쉼표는 **필수가 아니다**. 호흡상 필요할 때만 쓰고 남용하지 않는다.

| | 과용(✗) | 권장(✓) |
|---|---|---|
| 접속부사+쉼표 | 따라서, 피고의 주장은, 이유 없다. | 따라서 피고의 주장은 이유 없다. |
| 접속부사+쉼표 | 그러나, 변제 사실은, 인정되지 않는다. | 그러나 변제 사실은 인정되지 않는다. |

### C-2. 큰따옴표(직접 인용) vs 작은따옴표(강조·요약 인용)

큰따옴표("")는 글 가운데 직접 대화·인용을 **그대로 옮길 때** 쓴다. 작은따옴표('')는 마음속 생각, 강조, 또는 인용한 말 안의 인용에 쓴다. 실무에서 요약·간접 인용에까지 큰따옴표를 쓰는 오용에 주의한다.

| | ✗ | ✓ |
|---|---|---|
| 직접 인용(원문 그대로) | 피고는 갚겠다고만 하였다 | 피고는 "다음 달에 갚겠다"라고 진술하였다 |
| 요약·간접 인용 | 증인은 "피고가 돈을 빌렸다고 들었다"고 진술 | 증인은 피고가 돈을 빌렸다는 말을 들었다고 진술하였다 |
| 강조 | 핵심은 "변제기 도래" 여부다 | 핵심은 '변제기 도래' 여부다 |
| 용어 정의·강조 | 이른바 "선의취득"이 문제된다 | 이른바 '선의취득'이 문제된다 |

- 화자의 말을 **토씨 하나까지 그대로** 옮길 때만 큰따옴표. 내용만 추려 전하는 간접화법은 따옴표 없이('~라고 진술하였다') 적는다.

### C-3. 괄호 안 마침표, 괄호 뒤 조사

괄호 안의 내용이 **온전한 문장**이면 그 안에 마침표를 찍고, 어구·단어 수준이면 찍지 않는다. 닫는 괄호 뒤에 조사가 이어질 때 괄호 앞뒤를 띄우지 않는다. 문장 끝의 괄호 주석은 닫는 괄호 **뒤**에 마침표를 찍는다.

| | ✗ | ✓ |
|---|---|---|
| 어구 괄호(마침표 X) | 손해를 배상하였다(2023. 5. 1.자.) | 손해를 배상하였다(2023. 5. 1.자) |
| 문장 괄호(안에 마침표) | 변제기가 지났다 (이 점은 다툼이 없다) | 변제기가 지났다.(이는 다툼이 없다.) → 변제기가 지났다(이는 다툼이 없다). |
| 문장 끝 주석 | 청구를 인용한다(주문 제1항 참조.) | 청구를 인용한다(주문 제1항 참조). |
| 괄호 뒤 조사 | 갑 제1호증 (차용증) 을 보면 | 갑 제1호증(차용증)을 보면 |
| 날짜 마침표 | 2024. 3. 2 자 | 2024. 3. 2.자 (연·월·일 모두 마침표) |

- 날짜를 마침표로 줄여 적을 때는 '일' 뒤에도 마침표를 찍는다('2024. 3. 2.'). 그 뒤에 '자·경·부' 등이 붙으면 마침표 뒤에 붙여 쓴다('2024. 3. 2.자').
- 문장 전체에 대한 괄호 주석은 마침표를 괄호 **밖**(닫는 괄호 뒤)에 찍는다.
