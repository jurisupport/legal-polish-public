# 문서 종류별 점검 포인트

문서 종류별로 윤문·검토 시 "무엇을 확인해야 하는가"를 점검하는 체크리스트입니다. 아래 예시는 모두 설명을 돕기 위해 창작된 가상 예이며, 실제 사건과 무관합니다.

## 1. 소장·답변서

법원에 청구의 내용과 그 근거를 처음 제시하거나(소장), 상대 청구에 대한 1차 방어선을 세우는(답변서) 문서다. 소송의 골격을 정하므로 청구취지·청구원인의 정합성이 핵심이다.

- [ ] 청구취지가 그대로 주문이 될 수 있을 만큼 특정·완결되어 있는가(금액·이행 대상·기간·이자 기산일·소송비용·가집행 선언 등 누락 없는가)
- [ ] 청구원인이 청구를 뒷받침하는 요건사실을 빠짐없이 담고 있는가(권리근거 규범의 요건과 1:1로 대응하는가)
- [ ] 답변서에서 상대 주장에 대해 인정·부인·부지·항변이 명확히 구분되어 있는가(뭉뚱그린 "전부 부인"으로 자백 간주 위험을 키우지 않았는가)
- [ ] 결론(청구취지·청구 인용/기각 구함)이 앞부분에 두괄식으로 제시되는가
- [ ] 다툼 없는 사실과 다투는 사실이 구분되어 쟁점이 자연스럽게 드러나는가
- [ ] 주장과 입증(증거)의 연결이 표시되어 있는가
- [ ] 항변은 항변대로 독립된 항목으로 정리되어 주장과 섞이지 않았는가

**윤문 시 자주 손볼 지점**
- 청구원인이 시간순 사실 나열에 그쳐 "그래서 무슨 권리가 발생했는가"가 흐려지는 경우 → 요건사실 단위로 소제목을 붙여 재배열. (가상 예: "원고는 …을 지급하였다. 그 뒤 피고는 …" 식 서술을 "① 계약 성립 ② 원고의 이행 ③ 피고의 불이행" 구조로 재구성)
- 답변서에서 인정·부인이 한 문장에 섞여 있는 경우 → 항목을 쪼개 각 사실별로 태도를 분명히.

## 2. 준비서면

소장·답변서 이후 쟁점을 좁혀가며 상대 주장을 반박하고 자기 주장을 보강하는 문서다. 변론의 실질적 공방이 이루어지는 곳이므로 쟁점 정리와 논리 전개가 생명이다.

- [ ] 이 서면이 다루는 쟁점이 서두에 명시되어, 무엇에 답하는 서면인지 분명한가
- [ ] 상대 주장을 정확히 요약한 뒤 반박하는 구조인가(상대 주장을 왜곡하거나 유리하게만 인용하지 않았는가)
- [ ] 각 주장에 대응하는 증거가 명시적으로 연결되어 있는가(사실 주장이 증거 없이 떠 있지 않은가)
- [ ] 논리 전개가 한 방향으로 흐르는가(같은 쟁점이 여러 군데 흩어져 반복되지 않는가)
- [ ] 핵심 주장이 부수적 논점에 묻히지 않고 비중에 맞게 배치되었는가
- [ ] 법리 주장과 사실 주장이 구분되어 있는가
- [ ] 결론 문단이 "그러므로 …하여야 한다"로 청구취지·신청 취지와 맞닿아 있는가

**윤문 시 자주 손볼 지점**
- 상대 주장 인용과 자기 반박이 한 문단에 엉켜 누구 말인지 헷갈리는 경우 → "상대방은 …라고 주장한다. 그러나 …" 식으로 인용과 반박을 시각적으로 분리.
- 감정적 수식어("터무니없는", "억지 주장")가 논거를 약화시키는 경우 → 사실·법리로 대체하고 수식어 제거.

## 3. 상소이유서

원심(1심·항소심) 판단의 어느 부분이 왜 잘못되었는지를 특정하여 다투는 문서다. 막연한 불복이 아니라 "원심의 이 판단을 이 이유로 취소·파기해 달라"는 조준이 핵심이다.

- [ ] 다투는 대상이 원심 판결의 어느 판단(어느 쟁점·어느 부분)인지 구체적으로 특정되어 있는가
- [ ] 불복 사유가 법리오해·사실오인·심리미진·이유불비 등으로 성질에 맞게 분류·정리되어 있는가
- [ ] 상고심의 경우 사실심 재판단을 구하는 듯한 주장(사실오인 단순 반복)에 머물지 않고 법령 위반 차원으로 정리되었는가
- [ ] 각 상소이유마다 "원심이 이렇게 판단했다 → 그런데 이것이 잘못이다 → 따라서 이렇게 되어야 한다"의 구조가 갖춰졌는가
- [ ] 원심 판결문의 해당 판시를 정확히 인용·적시하였는가(원심이 하지 않은 판단을 했다고 오인하지 않았는가)
- [ ] 결론에서 구하는 재판(파기환송·자판 등)이 명시되어 있는가
- [ ] 여러 상소이유가 중복·포함 관계 없이 정리되었는가

**윤문 시 자주 손볼 지점**
- 1심에서 한 주장을 그대로 복사해 "원심이 틀렸다"는 연결 고리가 빠진 경우 → 원심 판시를 끌어와 "원심은 …라고 보았으나"로 다리를 놓아 상소이유의 형식을 갖춤.
- 법리오해 주장 안에 사실오인이 뒤섞인 경우 → 성질별로 항목을 분리.

## 4. 증인신문사항

증인으로부터 입증하려는 사실을 끌어내기 위해 미리 작성하는 질문 목록이다. 한 문항에 한 사실을 묻고, 입증취지와 연결되어야 효과적이다.

- [ ] 각 문항이 단일 쟁점에 초점을 맞추는가(사실·시점·평가를 한 질문에 뭉쳐 답변의 증거가치를 떨어뜨리는 복합질문이 되지 않았는가)
- [ ] 주신문에서 답을 미리 담은 유도신문이 되어 있지 않은가(이의 대상이 될 표현을 피했는가)
- [ ] 각 문항(또는 묶음)이 어떤 입증취지에 봉사하는지 분명한가(입증과 무관한 군더더기 질문이 없는가)
- [ ] 질문이 시간·사건 순서 등 따라가기 쉬운 흐름으로 배열되었는가
- [ ] 증인이 직접 경험·인식한 사항을 묻는가(전문·추측·의견을 구하는 질문이 아닌가)
- [ ] 답이 "예/아니오"로 닫혀버려 정작 듣고 싶은 진술을 막지 않는가(개방형/폐쇄형이 목적에 맞는가)
- [ ] 결정적 사실은 다른 증거·정황으로 미리 토대를 깐 뒤 묻도록 배치되었는가

**윤문 시 자주 손볼 지점**
- "증인은 그날 피고가 돈을 빌려달라고 하면서 다음 달에 갚겠다고 말한 것을 들었지요?"처럼 사실 여럿과 결론을 한 문항에 욱여넣은 경우(가상 예) → "그날 누가 있었나요" "무슨 대화가 오갔나요" 식으로 쪼개 증인 입에서 사실이 나오게.
- 의견·평가를 구하는 질문 → 경험한 사실을 묻는 형태로 전환.

## 5. 형사 변론요지서·의견서

피고인을 위해 공소사실에 대한 법적·사실적 방어와 양형 의견을 정리하는 문서다. 구성요건 해당성·위법성·책임의 단계와 양형사유가 명확히 구분되어야 한다.

- [ ] 무죄·다툼을 구하는 부분과 인정하되 양형을 구하는 부분이 명확히 갈라져 있는가
- [ ] 다투는 경우 구성요건 해당성 → 위법성 → 책임의 범죄성립 단계 중 어디를 다투는지 분명한가(단계를 건너뛰거나 뒤섞지 않았는가)
- [ ] 공소사실을 정확히 적시한 뒤 그에 대응해 반박하는가
- [ ] 증거능력·증명력에 관한 다툼(위법수집증거, 전문법칙 등)이 사실 다툼과 구분되어 있는가
- [ ] 양형사유가 유리한 정상(반성·합의·초범·부양가족 등)을 사실관계와 함께 구체적으로 제시하는가(추상적 선처 호소에 그치지 않는가)
- [ ] 양형 부분과 유무죄 부분의 논조가 모순되지 않는가(전면 부인하면서 동시에 깊이 반성한다고 하지 않는가)
- [ ] 결론에서 구하는 바(무죄·선고유예·집행유예·감경 등)가 명시되어 있는가

**윤문 시 자주 손볼 지점**
- 사실관계 서술과 법적 평가가 한 덩어리로 흐르는 경우 → "사실관계 — 법적 평가(해당성/위법성/책임) — 양형"으로 층을 나눠 재배치.
- 양형사유가 "피고인은 깊이 반성하고 있다" 같은 추상적 문장에 그친 경우(가상 예) → 합의 경위·피해 회복액·부양 상황 등 구체적 사실로 뒷받침.

## 6. 법률의견서

의뢰인의 질의에 대해 법적 결론과 근거, 그 한계를 정리해 제시하는 자문 문서다. 질의→결론의 두괄식 구성과 전제사실·근거·가정의 명시가 핵심이다.

- [ ] 검토 대상 질의(쟁점)가 서두에 분명히 정리되어 있는가
- [ ] 결론이 앞부분에 두괄식으로 제시되는가(읽는 사람이 끝까지 가야 답을 알게 되지 않는가)
- [ ] 결론을 떠받치는 전제(사실관계·가정)가 어디까지인지 선을 그어 두었는가(그 전제가 바뀌면 결론도 재검토 대상임이 드러나게)
- [ ] 결론의 근거가 되는 법령·판례·해석이 제시되어 있는가
- [ ] 단정할 수 있는 부분과 다툼·변동 여지가 있는 부분의 경계가 표시되어 있는가(확정/조건부/가능성의 강도가 구분되는가)
- [ ] 의뢰인이 취할 수 있는 선택지·후속 조치가 있다면 정리되어 있는가
- [ ] 자문의 용도·범위·한정(이 의견서가 다루지 않는 사항)이 분명한가

**윤문 시 자주 손볼 지점**
- 결론이 글 맨 끝에 한 줄로 숨어 있는 경우 → 서두에 "결론: …"을 두괄식으로 끌어올리고, 본문은 그 근거로 정렬.
- "문제될 것으로 보인다", "어려울 수 있다"처럼 결론의 강도가 모호한 경우(가상 예) → 확정/가능성/조건부를 구분해 "…인 한 적법하다", "다툼의 여지가 있으나 …일 가능성이 높다" 식으로 강도를 명시.
